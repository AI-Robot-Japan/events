# Claude Code Setup / Claude Code セットアップ

This guide explains how to use Claude Code with this repository.
このガイドでは、このリポジトリでClaude Codeを使用する方法について説明します。

## What is Claude Code? / Claude Codeとは？

Claude Code is Anthropic's AI-powered coding assistant that helps you write, understand, and debug code more efficiently.
Claude Codeは、コードの記述、理解、デバッグをより効率的に行うためのAnthropicのAI駆動型コーディングアシスタントです。

## Prerequisites / 前提条件

- Claude Code or compatible AI coding assistant / Claude Codeまたは互換性のあるAIコーディングアシスタント
- Git installed / Gitがインストールされていること
- Basic understanding of the project structure / プロジェクト構造の基本的な理解

## Configuration Files / 設定ファイル

This repository includes the following configuration files for Claude Code:
このリポジトリには、Claude Code用の以下の設定ファイルが含まれています：

### `.clinerules`

This file contains guidelines and rules for AI-assisted coding in this project.
このファイルには、このプロジェクトでのAI支援コーディングのガイドラインとルールが含まれています。

Key sections include:
主なセクションは以下の通り：

- **Project Context** - Overview of the repository purpose / リポジトリの目的の概要
- **Coding Guidelines** - Best practices for writing code / コードを書くためのベストプラクティス
- **Documentation Standards** - How to document your code / コードのドキュメント化方法
- **Testing Requirements** - Testing expectations / テストの期待事項
- **Security Guidelines** - Security best practices / セキュリティのベストプラクティス

## Using Claude Code / Claude Codeの使用

### 1. Clone the Repository / リポジトリのクローン

```bash
git clone https://github.com/AI-Robot-Japan/events.git
cd events
```

### 2. Open with Claude Code / Claude Codeで開く

Open the repository in your editor with Claude Code enabled.
Claude Codeが有効になっているエディタでリポジトリを開きます。

### 3. Verify Configuration / 設定の確認

Claude Code will automatically detect the `.clinerules` file and apply the guidelines.
Claude Codeは自動的に`.clinerules`ファイルを検出し、ガイドラインを適用します。

### 4. Start Coding / コーディングを開始

You can now use Claude Code to:
Claude Codeを使用して以下を行えます：

- Generate code snippets / コードスニペットを生成
- Explain existing code / 既存のコードを説明
- Suggest improvements / 改善を提案
- Debug issues / 問題をデバッグ
- Write tests / テストを書く

## Best Practices / ベストプラクティス

1. **Review AI Suggestions** / AI提案をレビュー
   - Always review and understand code generated by AI
   - AIによって生成されたコードは常にレビューして理解する

2. **Follow Project Guidelines** / プロジェクトガイドラインに従う
   - Ensure AI-generated code follows the `.clinerules` guidelines
   - AI生成コードが`.clinerules`ガイドラインに従っていることを確認

3. **Test Thoroughly** / 徹底的にテスト
   - Test all AI-generated code before committing
   - コミット前にすべてのAI生成コードをテスト

4. **Document Changes** / 変更をドキュメント化
   - Document any significant changes or additions
   - 重要な変更や追加をドキュメント化

## Troubleshooting / トラブルシューティング

### Claude Code not detecting .clinerules

If Claude Code is not applying the rules:
Claude Codeがルールを適用していない場合：

1. Ensure the `.clinerules` file is in the root directory
2. Restart your editor
3. Check Claude Code settings

### Issues with AI Suggestions

If you're getting unexpected suggestions:
予期しない提案が表示される場合：

1. Review the `.clinerules` file
2. Provide more context in your prompts
3. Be specific about what you need

問題や質問がある場合：
